Actions

Details

Clean HTML

  • Removes duplicate tags
  • Removes leading and trailing spaces
  • Removes extra spaces
  • Moves </strong> tags before <br> tags
  • Removes MS Word comments
  • Removes track changes coding
  • Removes empty tags
  • Adds &nbsp; when necessary
    • Before a colon, quote or $ in French
    • Dates
  • Cleans up tables
    • Removes deprecated attributes
    • Adds WET classes
  • Removes <u> tag from <a>
  • Removes type on lists
  • Will turn 2 lists into 1 if one after the other
  • Removes <strong> from headings
  • Adds <h2> to Table of contents
  • Fixes French characters
  • Add.lst-spcd to <ul> or <ol> if content in at least 1 <li> contains more than 150 characters
  • Removes <p> tag from <li>, <td>, <th>, <dt> and <dd> if there is only 1 <p> tag
  • Formats code

Footnotes

Applies the WET footnotes styles

Canada.ca links

Converts "https://www.canada.ca/" to "/content/canadasite/"

Note: only use when the page will be hosted on Canada.ca

H2 to details/summary

  • Adds <details> and <summary> tags to H2 sections
  • Adds expand all / collapse all buttons

Remove UTM codes

  • Removes UTM codes from links that accidentally contains them
